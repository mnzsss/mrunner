## [0.0.0-beta.1] - 2025-12-23

### Features

- **updater:** add auto-update infrastructure with signing
- **core:** monorepo restructure with apps/packages organization
- **bookmarks:** buku integration for bookmark management
- **ui:** command palette interface with keyboard navigation
- **plugins:** extensible plugin system

### Infrastructure

- GitHub Actions workflow for cross-platform releases (Linux, Windows)
- AUR package support for Arch Linux
- Code signing for secure auto-updates
