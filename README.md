# MRunner

**A fast, keyboard-first application runner for Linux**

MRunner is a fast, keyboard-driven application runner designed for Linux users who prefer speed, simplicity, and minimalism.

## Bookmarks

MRunner includes built-in bookmark support to speed up your daily workflow.

Create and access bookmarks directly from the runner to open:

- Frequently used websites
- Local directories
- Common resources

Everything is accessible in seconds ‚Äî no browser hunting or menu digging.

---

## Philosophy

MRunner is built with a clear goal: **stay simple**.

It focuses on the most common tasks performed every day and avoids unnecessary features, heavy customization layers, or complex configuration files.

If you live on the keyboard, MRunner is designed for you.

## Installation

### Arch Linux (AUR)

Install using an AUR helper:

`yay -S mrunner`

Or use your preferred AUR helper.

## Usage

Open MRunner using your configured shortcut and start typing:

- Application names
- Folder paths
- Bookmark names
- Common commands

Results appear instantly as you type.
